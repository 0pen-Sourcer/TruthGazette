Thank you for your interest in contributing to The Truth Gazette!

Please note: This project contains server-side secrets and rate-limit configuration. Do not commit API keys or credentials.

Contribution guidelines:
- Fork the repository and open a PR to the `main` branch.
- Describe your change clearly and include tests where applicable.
- For security-sensitive changes (auth, rate limiting, key handling), open an issue first and discuss the design.

Code style
- Keep JS code clean and documented. Add comments to server-side prompt changes.
